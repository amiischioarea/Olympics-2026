# Olympics-2026 Management System
A full-stack web application designed to manage and visualize data for the 2026 Olympic Games. 
This project provides a clean interface for tracking athletes, events, and results.

# Project Structure
The repository is organized into two main components:
  - frontend: contains the user interface 
  - backend: the core of the application

# Key Functionalities
The application allows users to interact with the Olympic database through the following features:
  - add new athletes
  - result tracking
  - data Persistence
